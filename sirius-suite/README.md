Each kernel contains:
- Makefile
- input (when applicable, larger inputs not included)
- baseline
- pthread/phi
- GPU

Each benchmark is run the following way:  
./executable ../input/input

Regex takes 2 inputs:  
./regex ../input/list ../input/questions
